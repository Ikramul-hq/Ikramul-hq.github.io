This portfolio is designed with a **Dark & Yellow High-Contrast Theme** to ensure readability and a modern aesthetic. It is a static site hosted on GitHub Pages, optimized for speed and performance.

### Key Features:
* **Responsive Design:** Fully adaptive layout for Desktop, Tablet, and Mobile.
* **Academic Timeline:** A custom vertical timeline displaying Education and Experience with aligned dates and grades.
* **Research Showcase:** Categorized publications (Journal, Conference, Workshop) with links to PDFs and code.
* **Project Cards:** Detailed cards for research and industrial projects with tech stack tags.
* **Sidebar Navigation:** A static, scrollable sidebar with quick access to contact info and a **Download CV** button.
* **Contact Form:** Integrated **Formspree** contact form for direct messaging without a backend server.
